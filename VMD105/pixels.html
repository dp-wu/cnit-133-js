Photoshop Final Project:

First VMD105 project using Photoshop. It is a selfies collage utilizing layers, cropping tools.

There is my grown-up self and my elementary school kid self.

:doodle { @grid: 15 / 100vmax; background: #fff; cursor: pointer; } :doodle(:hover) { --s: 1 ; } :after { content: @pick(y, W); font-size: 4vmax; color: #FAF5E4; transform: scale(@rand(.1, .9)); } transition: .5s cubic-bezier(.175, .885, .32, 1.275); transition-delay: @rand(1500ms); transform: translateY(calc(var(--s) * 150%)) rotate(calc(var(--s) * 360deg));
